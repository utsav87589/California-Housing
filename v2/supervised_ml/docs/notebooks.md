### ------------ Folder name : notebooks --------------------

- The 'notebooks' folder inside tells a unique story of where the certain actions and decisions were made on the data alongside the 'src' folder.

- They have the core logic of data being transformed and the logic being implemented to a fully functional model, that performs the real job.

- Inside the folder I have the following files, each one with unique responsiblity : 

1. basic_operations : loading the raw data, earliest possible split, checking out basic things

2. predictions : predictions on the clean data, saving the models, making the graphs and the comparison metrices

3. process_train : featured engineering and EDA on train data

4. process_valid : featured engineering and EDA on valid data
